This project is an ESP32 firmware designed to fetch data from a solar panel sensor and transmit it to the Azure IoT cloud using an API. \begin{DoxyItemize}
\item {\bfseries{Sensor Data Retrieval\+:}} The firmware is capable of reading data from the connected solar panel sensor and additionnal sensors such as a temperature sensor.
\item {\bfseries{Azure IoT Integration\+:}} It securely sends the collected data to the Azure IoT cloud via API, providing a seamless way to monitor and analyze solar panel performance.
\item {\bfseries{Power Efficiency\+:}} The firmware is optimized for power efficiency to ensure minimal energy consumption, making it suitable for battery-\/powered applications.
\item {\bfseries{Configurability\+:}} The firmware allows easy configuration of parameters such as Wi\+Fi credentials, Azure IoT connection details, and sensor communication settings through a uart connection.
